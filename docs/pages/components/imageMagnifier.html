ImageMagnifier

Component to create a magnify effect on ImageView on onMouseEntered event.

ImageMagnifier class is generic in T extends ImageView.

Constructors

	public ImageMagnifier(T image, double magnifyAmp, boolean keepOldHandlers)

  • image : ImageView in which magnify effect will be mounted.
  • magnifyAmp : amplitude of magnify effect.
  • keepOldHandlers : indicates if old EventHandlers are to keep or discard.

There are more constructors with different parameters, but all will internally call this first one.

Variations :

  • Constructors that don't receive keepOldHandlers this value is assumed as true.
  • Constructors that don't receive magnifyAmp this value is assumed as 1.5.
  • Constructors that receive IComponent<T> component obtain ImageView from component.getNode().
