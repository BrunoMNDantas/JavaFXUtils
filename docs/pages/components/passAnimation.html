JavaFX Utils

PassAnimation

Component to help implement animation based on onMouseEnter and onMouseExit as entry and exit points respectively.

Constructors

	public PassAnimation(T node,
		EventHandler<? super MouseEvent> onEnter,
		EventHandler<? super MouseEvent> onExit,
		boolean keepOldHandlers)

  • node : Node in which will be mounted animation.
  • onEnter : entry point of animation.
  • onExit : exit point of animation.
  • keepOldHandlers : indicates if oldHandlers are to keep or discard.

There are more constructors with different parameters, but all will internally call this first one.

Variations :

  • Constructors that don't receive keepOldHandlers this value is assumed as true.
  • Constructors that receive IComponent component obtain Node from component.getNode().
