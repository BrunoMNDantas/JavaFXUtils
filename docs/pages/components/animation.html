JavaFX Utils

Animation

Component to help implement animation triggered by mouse events, this animation has an entry and an exit point newEnterHandler e newExitHandler respectively.

This component helps implementation of components as:

  • PassAnimation
  • PressAnimation
  • PressAndPassAnimation

Constructors

	public Animation(	T node,
		EventHandler<? super MouseEvent> oldEnterHandler,
		EventHandler<? super MouseEvent> newEnterHandler,
		EventHandler<? super MouseEvent> oldExitHandler,
		EventHandler<? super MouseEvent> newExitHandler,
		Consumer<EventHandler<? super MouseEvent>> enterSetter,
		Consumer<EventHandler<? super MouseEvent>> exitSetter,
		boolean keepOldHandlers)

  • node : Node in which will be mounted animation.
  • oldEnterHandler : old entry point to chain with new one.
  • newEnterHandler : entry point of animation.
  • oldExitHandler : old exit point to chain with new one.
  • newExitHandler : exit point of animation.
  • enterSetter : setter for new entry point.
  • exitSetter : setter for new exit point.
  • keepOldHandlers : indicates if old EventHandlers are to keep or discard.

There are more constructors with different parameters, but all will internally call this first one.

Variations :

  • Constructors that don't receive keepOldHandlers this value is assumed as true.
  • Constructors that receive IComponent<T> component obtain Node from component.getNode().
