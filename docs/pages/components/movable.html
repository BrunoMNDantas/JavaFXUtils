JavaFX Utils

Movable

Component to help implement movable Node which can be dragger around.

This component sets EventHandlers on onMousePressed and onMouseDragged events and change the position of Node.

If during the mounting process any Exception occurs, ComponentException will be thrown.

Constructors

public Movable(T node, Consumer<MouseEvent> onMove, boolean keepOldHandlers)

  • node : Node in which will be mounted the movable functionality.
  • onMove : action to be triggered on mode.
  • keepOldHandlers : indicates if the EventHandlers are to keep or to be discarded.

There are more constructors with different parameters, but all will internally call this first one.

Variations :

  • Constructors that don't receive boolean keepOldHandlers this value is assumed as true.
  • Constructors that receive BiConsumer onMove action called on move that receives x and y coordinates.
  • Constructors that receive IComponent<T> component and obtain Node object from component.getNode().

Usage

	Button button;

	try {
		new Movable<>(button).mount();
	} catch (ComponentException e) {
		//Treat Exception
	}
